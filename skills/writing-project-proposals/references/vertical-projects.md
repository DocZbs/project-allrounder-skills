# Vertical-Project Routes

Always use the current official call and institutional notice. The route determines questions, not fixed answers.

## Natural-Science Programs

Check category, eligibility/limits, scientific question, significance, novelty, objectives, research content, methods, technical route, preliminary evidence, feasibility, team/resources, ethics/data, indicators, schedule, and budget. Verify the current sponsor's generative-AI policy before producing formal application text, progress reports, or closeout/output reports.

Do not manufacture preliminary data or treat expected results as observations.

## Social-Science and Humanities Programs

Check the current project class, main discipline and cross-disciplinary route, topic rules, duplicate applications, anonymized/non-anonymized materials, problem and perspective, literature and prior awards, conceptual or methodological contribution, evidence base, outputs, duration, budget, and institutional review.

Keep political/academic direction, truthfulness, participant consent, authorship, and duplicate-research checks explicit where applicable.

## Key R&D and Mission-Oriented Programs

Decompose guide direction → project objective → topic/task → indicator → participating unit → milestone → deliverable → demonstration/acceptance evidence. Check consortium roles, task books, funds allocation, matching resources, data/reporting, engineering maturity, demonstration conditions, and performance evidence.

## Provincial, Municipal, and Institutional Programs

Treat the local call, online-system outline, institution notice, and current finance rules as controlling. Reuse the common argument and evidence framework, not assumed national-program limits.

## Vertical Submission Audit

- exact program/category/year confirmed;
- applicant/team eligibility and duplicate/limit checks confirmed;
- official form and system outline version confirmed;
- every required field and attachment mapped;
- facts, references, and preliminary work verified;
- targets have baseline, method, and feasibility support;
- budget matches allowed categories and actual work;
- ethics, security, confidentiality, data, and IP reviewed;
- responsible applicant completes required authorship, verification, declaration, signatures, and submission.
