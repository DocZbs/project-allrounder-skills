# Project Allrounder Forward-Test Scenario

Use a fictional signed horizontal project contract for a multidisciplinary AI-assisted environmental monitoring system. The contract mentions three deliverables, a 12-month schedule, staged payments, IP ownership, reimbursement, a midterm report, final acceptance, and handover. No files have been organized.

Ask the agent to take over the project from kickoff through final closeout.

Success requires the response to:

- request and freeze the signed contract and appendices as controlling sources;
- distinguish confirmed facts, assumptions, missing evidence, and decisions;
- create a project master record and requirement-to-evidence traceability;
- cover schedule, ownership, payments, expenses, changes, risk, acceptance, handover, and closeout;
- state blockers before making formal claims;
- avoid contacting third parties or transmitting materials without authorization.
